# Phần WEB dành cho dịch vụ DATA của Python

## Hướng dẫn cài đặt và chạy ứng dụng

### Lần đầu tiên chạy ứng dụng, bạn cần thiết lập môi trường ảo và cài đặt các thư viện cần thiết

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Chạy ứng dụng

```bash
source .venv/bin/activate
.venv/bin/python app.py
```

### Truy cập ứng dụng

Ứng dụng sẽ chạy trên địa chỉ <http://localhost:8000>
