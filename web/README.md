# Hướng dẫn

- Nếu lần đầu: Chạy npm install để cài đặt các gói phụ thuộc.
- npm run dev để chạy ứng dụng trong môi trường phát triển.
- Mở trình duyệt và truy cập <http://localhost:5173> để xem ứng dụng

## Triển khai trên môi trường máy chủ

- Chạy npm run build để tạo bản dựng sản xuất.
- Triển khai thư mục `dist` lên máy chủ của bạn.

## Cấu trúc dự án

- main.tsx khai báo routes

## Khai báo tạo MENU

- Vào src/App.tsx
